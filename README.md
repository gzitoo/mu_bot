# 🤖 Bot Auto Reset MU Online

Script simples em Python para automação de reset no MU Online usando **PyAutoGUI**.

> ⚠️ Use por sua conta e risco. Automação pode gerar ban em servidores.

---

## 📌 Pré-requisitos

### 🐍 1) Instalar o Python

Faça o download no site oficial:

👉 https://www.python.org/downloads/

Durante a instalação, **marque obrigatoriamente**:
- ✅ Add Python to PATH
- ✅ pip

---

### 🧰 2) Instalar a biblioteca necessária

Abra o terminal (CMD, PowerShell ou terminal do VS Code) e execute:

```bash
python -m pip install pyautogui
```

---

## 🖥️ Configuração do ambiente

- 🔐 **Execute o VS Code como administrador**
- 🎮 Abra o MU Online
- 🖱️ Deixe a janela do jogo em foco

---

## ▶️ Como usar

1. Abra a pasta do projeto
2. Execute o script:

```bash
python mu_bot.py
```

3. Clique na janela do MU quando o script iniciar
4. O bot irá:
   - Distribuir os pontos **uma única vez**
   - Entrar em loop infinito tentando `/reset`
